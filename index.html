WHITE FOX IRRIGATION
Use this page instead of ad hoc Serial helpers. Live data uses the status text poll + read for settings.
BLE TROUBLESHOOTING
If connect shows nothing, the phone is not seeing the receiver advertiser. On USB Serial (115200) you should see BLE advertising as 'WhiteFox Irrigation' after Boot complete. If that line never appears, fix power/USB, wrong sketch/port, or a boot crash before BLE. If you upgraded Arduino ESP32, try Tools → Bluetooth mode (Bluedroid vs NimBLE). Latest receiver firmware (ESP32_Now_Protocol_Recieve.ino) disables WiFi modem sleep and periodically refreshes advertising when disconnected.
MASTER SHUTOFF
Settings → MASTER SHUTOFF / SYSTEM RESUME: tap to send master shutoff hold (pump off, all starts blocked) or master shutoff release. BLE disconnect clears the lock on the ESP32.
WATERING RULES
1) Normal watering: Soil below moisture threshold, inside time window, and cooldown clear.
2) Emergency watering: Soil below emergency threshold, ignores time window. Same cooldown as normal runs — no emergency start until cooldown has expired.
3) Manual watering: WATER ON clears cooldown and starts immediately; full cooldown applies from pump stop.
4) Cooldown: Any new watering run (auto, emergency, or manual) clears the prior lockout timer on the controller.
5) Flow guard: After a prime period, if smoothed flow stays below the firmware minimum while the pump is on, the run aborts (tune constants in ESP32_Now_Protocol_Recieve.ino).
6) Dry bench test: Settings → Advanced → 15 FLOW CHECK ENABLED = OFF (0) disables that abort; timer stop still applies. Turn ON (1) for normal use; SAVE CONFIG to persist.
TIME WINDOW
Use 24-hour values.
Examples: 07 = 7am, 14 = 2pm.
For 11am to 2pm set:
05 WATER WINDOW START = 11
06 WATER WINDOW END = 14

LONG-TERM HISTORY
NVS history on the ESP32 holds litres + run counts plus the last 5 Recent Runs and last 5 Event Logs. Opening History sends history; the ESP replies WF HT with totals, then WF RH/WF EH rows. CLEAR STORED sends clear history (reply OK history cleared).
